# Прогноз погоды

Веб-приложение для отслеживания прогноза погоды - учебный проект. В рамках данной работы я разработала приложение с интуитивно понятным интерфейсом для просмотра актуального прогноза погоды на основе геолокации пользователя или выбранных городов.

## Технические требования

В работе был использован чистый JavaScript + HTML и CSS без фреймворков. Для получения данных о погоде используется WeatherAPI. Сайт размещен на GitHub Pages.

## Дополнительные улучшения

### 1. Улучшенный UX/UI дизайн

- Градиентный фон с фиолетовыми оттенками
- Анимация появления модального окна
- Анимированные точки при загрузке

### 2. Оптимизация производительности

- Debounce для поиска городов (500ms)
- Параллельная загрузка погоды для нескольких городов
- Cache busting при обновлении данных

### 3. Детализированная информация

- Температура и ощущаемая температура
- Влажность и скорость ветра
- Иконки погодных условий
